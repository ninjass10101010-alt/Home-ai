## 🎨 Design Features Detailed

### Light/Dark/System Themes
- **Dark Mode:** Deep navy backgrounds, light text, glass blur effects
- **Light Mode:** Clean white backgrounds, dark text, subtle glass effects
- **System Mode:** Auto-detects OS preference, respects prefers-color-scheme
- **Persistent:** Saves to localStorage, restores on next visit
- **Instant Switching:** No page reload, smooth 0.3s color transition

### Theme Toggle Button
- **Always Visible:** Top-right of TopBar
- **Clear Icons:** Sun = light mode available, Moon = dark mode available
- **Accessible:** aria-label, keyboard navigable, 44×44px touch target
- **Animated:** Icon rotates 180° on click
- **Instant:** Changes apply immediately to all elements via CSS variables

### Accent Color Picker (6 Colors)
- **Nori (Blue)** — Default, primary brand
- **Violet** — Premium, special events
- **Rose** — Urgent, alerts
- **Cyan** — Information, secondary
- **Mint** — Success, completed
- **Amber** — Warnings, pending
- **Live Preview:** See color changes instantly
- **Persistent:** Saves to localStorage

### Settings Page with Theme Customization
1. **Profile Section** — Avatar, name, email, edit/sign out
2. **Theme Settings** ← MAIN THEME FEATURE
   - Mode selector (Light/Dark/System)
   - Accent color picker (6 swatches)
   - Live preview card
   - High contrast toggle
3. **Notifications** — 5 toggles
4. **Calendar & Integrations** — Connected calendars
5. **About** — Version, links, update check

### Glass Morphism (3 Variants)
- **Standard Glass** — 50% opacity, 20px blur, 8% border
- **Strong Glass** — 75% opacity, 20px blur, 30% border
- **Subtle Glass** — 35% opacity, 12px blur, 5% border
- **Dark/Light Aware** — Automatically switches opacity/blur for theme
